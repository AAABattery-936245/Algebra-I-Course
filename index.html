Lesson 1: Order of Operations

What is the Order of Operations?

When a math expression has more than one operation, we follow a set order so everyone gets the same answer. This order is often remembered with PEMDAS: Parentheses, Exponents, Multiplication and Division (left to right), Addition and Subtraction (left to right).

Example: Evaluate \[3 + 4 \times 2\].

  1. Multiply first: \[4 \times 2 = 8\]
  2. Add: \[3 + 8\]
  3. Result: 11

Practice Checkpoint

Question 1: Evaluate \[10 - 2 \times 3\].

  • Option A: 4
  • Option B: 24
  • Option C: 8

Hint: Multiply before you subtract.

Lesson 2: Working with Integers

What are Integers?

Integers are whole numbers that can be positive, negative, or zero. When adding or subtracting integers, it helps to think of a number line.

Example: Evaluate \[-5 + 8\].

  1. Start at -5 on the number line.
  2. Move 8 spaces to the right.
  3. Result: 3

Practice Checkpoint

Question 2: Evaluate \[-4 - 6\].

  • Option A: 2
  • Option B: -2
  • Option C: -10

Hint: Subtracting a positive number moves you further left on the number line.

Lesson 3: Fractions, Decimals, and Percents

Converting Between Forms

Fractions, decimals, and percents are three ways of writing the same value. To change a fraction to a decimal, divide the numerator by the denominator. To change a decimal to a percent, multiply by 100.

Example: Convert \[\frac{1}{4}\] to a decimal and a percent.

  1. Divide: \[1 \div 4 = 0.25\]
  2. Multiply by 100: 25%

Practice Checkpoint

Question 3: What is \(\frac{1}{5}\) written as a percent?

  • Option A: 5%
  • Option B: 20%
  • Option C: 50%

Hint: First divide 1 by 5 to get the decimal, then multiply by 100.

Lesson 4: Introduction to Ratios

A ratio compares two quantities. For example, if a recipe uses 2 cups of flour for every 1 cup of sugar, the ratio of flour to sugar is written \[2:1\].

Ratios help set the stage for the variables and equations you will see in Algebra I.

Lesson 1: Introduction to Variables

What is a Variable?

In algebra, a variable is a letter or symbol that stands for an unknown number. We often use x, but variables can also be y, a, or b.

For example, the expression \[x + 5\] means: “Some unknown number plus five.”

Evaluating Expressions

When we evaluate an expression, we replace the variable with a specific number. This process is called substitution.

Example: Evaluate \[3x + 2\] when \(x = 4\).

  1. Replace x with 4: \[(3 \times 4) + 2\]
  2. Multiply: \[12 + 2\]
  3. Add: 14

Practice Checkpoint

Question 1: Evaluate \[5x - 3\] if \(x = 3\).

  • Option A: 12
  • Option B: 15
  • Option C: 2

Hint: First multiply 5 times 3, then subtract 3.

Lesson 2: Solving One-Step Equations

What is an Equation?

An equation shows that two expressions are equal. It always uses an equals sign (\(=\)).

Example: \[x + 4 = 10\]. This means some number plus four equals ten.

Using Inverse Operations

To solve an equation, isolate the variable by undoing the operation around it. Use the opposite operation.

  • Addition → subtraction
  • Subtraction → addition

Remember: whatever you do to one side of the equation, do to the other side too.

Example

Solve \[x + 7 = 15\].

  1. Subtract 7 from both sides: \[x = 8\]

Practice Checkpoint

Question 2: Solve \[y - 3 = 12\].

  • Option A: \(y = 9\)
  • Option B: \(y = 15\)
  • Option C: \(y = 36\)

Hint: The inverse of subtracting 3 is adding 3.

Lesson 3: Solving Two-Step Equations

What is a Two-Step Equation?

A two-step equation needs two separate inverse operations to solve for the variable.

Example: \[2x + 3 = 11\].

First undo addition or subtraction, then undo multiplication or division.

Step-by-Step Example

  1. Subtract 3 from both sides: \[2x = 8\]
  2. Divide both sides by 2: \[x = 4\]

Practice Checkpoint

Question 3: Solve \[3x - 4 = 11\].

  • Option A: \(x = 5\)
  • Option B: \(x = 3\)
  • Option C: \(x = 7\)

Hint: First undo subtracting 4, then undo multiplying by 3.

Lesson 4: Connecting Expressions and Equations

Expressions are math phrases, like \[4x + 1\]. Equations are complete sentences that use \(=\), like \[4x + 1 = 13\].

When we solve equations, we find the value of the variable that makes both sides equal.

With practice, you can use the same process for many algebra problems.

Lesson 5: Solving Multi-Step Equations

A multi-step equation requires more than two operations to solve. Often, you must simplify the expression first by combining like terms before isolating the variable. Like terms are terms that share the exact same variable.

Example: Solve 4x + 2x - 5 = 13.
1. Combine like terms: 6x - 5 = 13
2. Add 5 to both sides: 6x = 18
3. Divide by 6: x = 3

Question 5: Solve 5x - 2x + 7 = 22.

  • A) x = 3
  • B) x = 5
  • C) x = 9

Hint: First combine 5x - 2x into 3x. Then subtract 7 from both sides.

Lesson 6: Variables on Both Sides

Sometimes, variables appear on both sides of the equals sign. To solve this, you must move all the variable terms to one side (usually the left) and all constant numbers to the other using inverse operations.

Example: Solve 7x = 3x + 16.
1. Subtract 3x from both sides: 4x = 16
2. Divide both sides by 4: x = 4

Question 6: Solve 8x = 2x + 24.

  • A) x = 3
  • B) x = 4
  • C) x = 6

Hint: Subtract 2x from both sides first to get 6x = 24.
